## Day 3 - User Directory

### Features Implemented
- Built a User Directory that fetches and displays 20 user profiles from the Random User API.
- Added search by name, sorting (A–Z and Z–A), filters for gender and country, and a favourite feature using Local Storage so favourite users remain saved after refreshing the page.

### Challenges Encountered
- The biggest challenge was making the search, sort, and filter features work together without affecting each other. I solved this by creating a single "applyFilters()" function to manage all of these features together.

### What I Learned
- Through this task, I learned how to work with API responses containing multiple user records, display data dynamically using DOM manipulation, use JavaScript array methods such as "map()" , "filter()" , and "sort()", handle user interactions with event listeners, and save data using Local Storage.
